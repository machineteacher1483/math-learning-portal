The Quadratic Formula

The quadratic formula is the universal solution to any quadratic equation. Memorise it, and you'll never be stuck on a quadratic again.

The Formula

Quadratic Formula

For any equation ax² + bx + c = 0:

x = (−b ± √(b² − 4ac)) / (2a)

This formula always gives the correct solutions — whether the quadratic factors nicely or not.

Example 1

Using the Formula

Solve 2x² − 5x + 2 = 0.

1. a = 2, b = −5, c = 2
2. x = (−(−5) ± √((−5)² − 4(2)(2))) / (2·2)
3. = (5 ± √(25 − 16)) / 4
4. = (5 ± √9) / 4
5. = (5 ± 3) / 4
6. x = (5+3)/4 = 2 or x = (5−3)/4 = 1/2

The Discriminant

b² − 4ac: How Many Solutions?

The expression b² − 4ac is called the discriminant (often written Δ). It tells you the number of real solutions before you fully solve.

Δ > 0

Two distinct real solutions

Δ = 0

One repeated real solution

Δ < 0

No real solutions

Example: For x² + 2x + 5 = 0: Δ = 4 − 20 = −16 < 0 → no real solutions.

Example: For x² − 6x + 9 = 0: Δ = 36 − 36 = 0 → one solution: x = 3.

Watch Out

Common Mistakes

  • Forgetting the ± — there are usually TWO solutions
  • Computing b² as (−b)² incorrectly — watch your signs
  • Dividing only the numerator, not the whole expression, by 2a
  • Always put the equation in standard form (= 0) before reading a, b, c

Your Turn

Try It Yourself

Q1. Solve x² + 4x − 5 = 0 using the quadratic formula.

Show Answer
a=1, b=4, c=−5
Δ = 16 + 20 = 36
x = (−4 ± 6) / 2
x = 1 or x = −5

Q2. Without fully solving, how many real solutions does 3x² − x + 2 = 0 have?

Show Answer

Δ = (−1)² − 4(3)(2) = 1 − 24 = −23 < 0. No real solutions.

Key Takeaways

1-Minute Summary

  • Formula: x = (−b ± √(b²−4ac)) / 2a
  • Discriminant Δ = b²−4ac: positive = 2 solutions, zero = 1, negative = none
  • Always put in standard form first
  • Don't forget the ±
